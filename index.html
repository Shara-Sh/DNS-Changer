DNS Changer is easy to use

Download, install, and change your DNS server settings in minutes

  1. Click on the Start button and type "Control Panel," then open it.
  2. Go to "Network and Sharing Center."
  3. Click on "Change adapter settings" on the left sidebar.
  4. Right-click on your active network connection and select "Properties."
  5. Choose "Internet Protocol Version 4 (TCP/IPv4)" and click "Properties."
  6. Select "Use the following DNS server addresses" and enter your preferred DNS server addresses.
  7. Click "OK" to save your changes.
